# Architecture Decision Records

i18n's design decisions are recorded as ADRs in the **planning repo**, not here —
one reviewed source of truth, maintained in one place.

- ADR workflow: <https://github.com/etamong-playground/planning/blob/main/wiki/concepts/docs-first-workflow.md>
- Fleet-wide ADRs (wiki concepts): <https://github.com/etamong-playground/planning/tree/main/wiki/concepts>
- Repo-scoped ADRs live under `planning/docs/adr/i18n/` as they are recorded.

Each code PR cites the specific ADR it implements. This file points at the ADR *home*, so
adding a new ADR never requires editing here.
